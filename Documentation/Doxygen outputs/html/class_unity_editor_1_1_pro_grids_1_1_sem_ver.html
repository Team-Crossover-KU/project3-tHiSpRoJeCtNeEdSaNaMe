Simple formatting for a version info. The following characters are available: 'M' Major 'm' Minor 'p' Patch 'b' Build 't' Lowercase single type (f, d, b, or p) 'T' Type 'd' Date 'D' Metadata 'R' The input string used to construct this VersionInfo. Escape characters with '\'.

ToString("\buil\d: T:M.m.p") returns "build: Final:2.10.1"
